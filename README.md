# routepix

## 機能

- 背景に地図が表示されます。
- 地図上に、GPXの軌跡がポリラインで表示されます。
- 軌跡上に、その場所で撮影された写真があることを示すピンが表示されます。
- ピンをクリックすると、撮影された写真がポップアップされます。
- ポップアップされた写真をクリックすると、別タブで拡大写真が表示されます。

## 実行準備

- src/assets/ に、GPXファイルと画像ファイルを置いてください。
- src/assets/ に置いたファイル名で、src/App.tsx を修正してください。
    - GPXファイルは1つ指定できます。
    - 画像ファイルは複数指定できます。

## 実行方法

- `npm install` で依存パッケージをインストールします。
- `npm run dev` で開発サーバーを起動します。
- `npm run build` でビルドします。
- `npm run preview` でビルドしたファイルをローカルでプレビューします。
